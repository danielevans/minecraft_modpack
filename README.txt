
INSTALLATION
==============================================

Ensure you have Java 7 (1.7)

Download and install Minecraft.

Use the launcher to run 1.7.10, then close the game.

Run the included Forge mod installer

After Forge installation is complete, Run the minecraft launcher and create a new profile
   - Set the game version to be the forge mod version (release 1.7.10-Forge10.13.0.1208)
   - Set the game directory to be a directory of your choice
   - Set the java executable to be the java 7 (1.7) executable

Copy the mods folder from this package into the directory you chose in the previous step

Start the game with that profile

Connect to the server

Profit.
